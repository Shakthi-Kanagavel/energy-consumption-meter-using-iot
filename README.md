# Smart Energy Consumption Monitoring System
##  Project Overview
The Smart Energy Consumption Monitoring System is an IoT-based project that monitors electricity usage in real time and predicts future energy consumption using Machine Learning.
The system collects energy data from smart sensors, stores the data in a database, displays real-time usage through a web dashboard, and uses ML algorithms to predict future consumption patterns.
---
##  Objectives
- Monitor real-time electricity consumption
- Store energy usage data securely
- Visualize consumption through a dashboard
- Predict future energy usage using Machine Learning
- Improve energy efficiency and reduce wastage

## ⚙️ Working Process
1. Energy sensors collect electricity consumption data.
2. NodeMCU sends collected data to the backend API.
3. Flask backend processes and stores data in MongoDB.
4. Dashboard displays real-time energy statistics.
5. Machine Learning model analyzes previous usage data.
6. Future energy consumption is predicted.
